## Issues to Fix

### Issue 1: Enum Confusion (HIGH PRIORITY)
**Problem**: `trust_boundary` uses different values than `trust_level`
- `TrustBoundary`: `trusted`, `partially_trusted`, `user_controlled`, `internal_trusted`
- `TrustLevel`: `trusted`, `untrusted`, `internal`

**Impact**: User confusion when writing topology YAML

**Fix**: Unify to single enum or add validation/aliases

### Issue 2: Tier 3 Not Implemented (HIGH)
**Problem**: Spec says Tier 3 = full swarm scan, but only tier 1-2 exist

**Fix**: Implement tier 3 with full swarm integration

### Issue 3: Token Cost Not Visible (MEDIUM)
**Problem**: Spec requires token cost visibility per component/agent

**Fix**: Add token tracking to scan results

### Issue 4: No Auto-Discovery (MEDIUM)
**Problem**: Spec mentions auto-discovery from OpenAPI/LangGraph

**Fix**: Add optional auto-discovery tooling
